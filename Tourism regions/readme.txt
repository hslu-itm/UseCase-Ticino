Just for info:

The files Lago_Maggiore_e_Valli.geojson, Lugano Region.geojson, Mendrisiotto.geojson and Bellinzona e Alto Ticino.geojson have been extracted from the ticinomap.geojson using the geojson viewer https://geojson.io/.

Steps:

1. load the ticinomap.geojson to https://geojson.io/
2. delete all the 'features' (other regions etc) that you are not interested in (click the feature and it gives you an option to delete it).
3. save the file to <region>_name.geojson using the  https://geojson.io/ save to file menu.
